2. Submit Extrinsic via Polkadot.js Apps
------------------------------------------------------------

# Steps:
1. Open https://polkadot.js.org/apps
2. Connect to Westend Asset Hub testnet
3. Select "Extrinsics" tab
4. Choose module: system
5. Choose call: remark
6. Paste the SHA‑256 hash string (e.g., 18de905f72c92a9fb895c6092ce6c01c9efb4c209fecc9853633adca6dd8575e)
7. Submit transaction using COS DEMO ACCOUNT (Talisman wallet)
8. Wait for block finalization
