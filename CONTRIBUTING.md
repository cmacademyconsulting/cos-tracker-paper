# Contributing Guidelines

Thank you for your interest in improving the COS™ Blockchain Tracker.

## How to Contribute
1. Fork the repository
2. Create a feature branch (`git checkout -b feature-name`)
3. Make your changes with clear commit messages
4. Submit a Pull Request (PR) describing your changes

## Reporting Issues
- Use the GitHub **Issues** tab to report bugs or suggest enhancements.
- Please include screenshots or Tx IDs if relevant.

## Ethics Reminder
All contributions must align with the COS™ ethics‑first methodology:  
Compliance · Oversight · Sustainability
